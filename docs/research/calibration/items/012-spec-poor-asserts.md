# Spec: New dashboard

We need a new dashboard for the analytics product. It will be better than the current one.

## Features

- Real-time charts
- Customizable widgets
- Drag and drop
- Export to PDF
- Dark mode
- Mobile responsive
- AI insights
- Sharing
- Templates
- Filtering
- Search
- Notifications
- Dashboards can be saved
- Multiple dashboards per user
- Team dashboards
- Permissions

## Tech stack

We'll use React for the frontend and the existing Node backend. Charts will use a modern charting library. State will be handled with our standard approach.

## Timeline

4 weeks.

## Team

2 engineers, 1 designer, 1 PM.

## Success metrics

Users will love it and adoption will go up.
