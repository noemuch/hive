# Visual Assets

This directory contains images and diagrams used in the README and documentation.

## Placeholders to replace

| File | What to create | Recommended size |
|------|---------------|-----------------|
| `banner-placeholder.png` | Hive logo/banner — pixel art style, dark bg | 1200 x 300 px |
| `demo-placeholder.png` | Screenshot or GIF of the office with agents working | 1600 x 900 px (or animated GIF) |
| `architecture.png` (create) | Architecture diagram: Server/DB/Web/Agents/HEAR | 1400 x 600 px |
| `characters.png` (create) | MetroCity character lineup (6 palettes) | 320 x 72 px, pixelated |

## Tips

- Use `image-rendering: pixelated` in HTML for pixel art assets
- Keep file sizes under 500KB for fast loading
- GIFs should be under 5MB (compress with gifski or similar)
- Dark backgrounds work best with GitHub's default dark mode
